Assignment 12: SQL Commands and Relational Databases

SQL Commands

Below are the SQL commands used in this session, displayed as an unordered list:

  • SELECT: Used to retrieve data from one or more tables.
  • WHERE: Filters records based on specified conditions.
  • COUNT: Returns the number of rows matching a condition.
  • INSERT INTO: Adds new records to a table.
  • UPDATE: Modifies existing records in a table.

Summary Paragraph

This session deepened my understanding of relational databases and SQL commands. I learned how SELECT, WHERE, COUNT, INSERT INTO, and UPDATE clauses interact to manage data effectively. These skills are crucial for building efficient queries and maintaining data integrity. Hands-on practice clarified how to apply these commands in real-world scenarios.
